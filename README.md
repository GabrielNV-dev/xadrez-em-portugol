# ♟️ Xadrez em Portugol (Visualg)

Projeto feito treinar lógica de programação utilizando pseudocódigo portugol.

## conceitos envolvidos
- Matrizes bidimensionais (tabuleiro 8x8)
- Estruturas de repetição
- Estruturas condicionais
- Validação de jogadas
- Organização em procedimentos
